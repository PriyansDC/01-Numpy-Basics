# NumPy Learning Journey 🚀

A beginner-to-intermediate NumPy learning repository created while following the **Chai Aur Code** NumPy series. This repository contains all the practice code, examples, and mini-projects completed during the learning process.

---

## 📚 Learning Roadmap

### 📌 Phase 1 — NumPy Foundations

Learned the basics of NumPy and array creation.

### Topics Covered

- Creating 1D and 2D arrays
- Python Lists vs NumPy Arrays
- Performance comparison
- `np.arange()`
- Creating arrays from scratch
  - `zeros()`
  - `ones()`
  - `full()`
  - `random()`
- Vectors, Matrices & Tensors
- Array Properties
  - Shape
  - Dimensions
  - Size
  - Data Type
- Reshaping Arrays
- Flatten vs Ravel
- Transpose

---

### 📌 Phase 2 — Array Operations

Performed common operations on NumPy arrays.

### Topics Covered

- 1D Slicing
- 2D Indexing & Slicing
- Sorting Arrays
- Boolean Filtering
- Masking
- Fancy Indexing
- `np.where()`
- Concatenation
- Shape Compatibility
- Vertical Stack (`vstack`)
- Horizontal Stack (`hstack`)
- Deleting Elements

### 📌 Phase 3 — Real World Business Example

Applied NumPy on a restaurant sales dataset.

### Operations Performed

- Total yearly sales
- Minimum sales
- Maximum sales
- Average sales
- Cumulative sales
- Data visualization using Matplotlib
- Vector Operations
  - Addition
  - Multiplication
  - Dot Product
  - Angle between vectors
- Vectorization
- Broadcasting

---

### 📌 Phase 4 — Saving Arrays & Image Processing

Worked with NumPy file storage and image manipulation.

### Topics Covered

- Saving arrays using `.npy`
- Loading NumPy arrays
- Reading image stored as NumPy array
- Converting NumPy logo into Dark Mode
- Displaying images using Matplotlib

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/PriyansDC/01-Numpy-Basics
```

Move into the project directory

```bash
cd numpy-learning
```

Create a virtual environment (Optional)

```bash
python -m venv venv
```

Activate it

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook

```bash
jupyter notebook
```
